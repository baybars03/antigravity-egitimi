# Antigravity Eğitimi - Yapılandırma Dosyaları 🚀

Bu depo, Google Antigravity IDE eğitim süreçlerinde ve modern geliştirme pratiklerinde ihtiyaç duyulan özelleştirilmiş yapılandırmaları içermektedir. Çalışma ortamını optimize etmek ve proje geliştirme süreçlerini hızlandırmak amacıyla hazırlanan özel **rules (kurallar)**, **skills (yetenekler)** ve **workflows (iş akışları)** bu dizinde toplanmıştır.

## 📂 Depo İçeriği

Eğitim boyunca kullanılacak olan yapılandırmalar üç ana kategoriye ayrılmıştır:

* **`/rules` (Kurallar):** Antigravity IDE'nin kodlama standartlarını, proje mimarisini ve genel davranış biçimlerini (örneğin; dil tercihleri, formatlama standartları) belirleyen temel yapılandırma dosyalarıdır.
* **`/skills` (Yetenekler):** Geliştirme sürecini destekleyen, spesifik görevleri yerine getirmek üzere tanımlanmış yapay zeka asistan yetenekleri ve özelleştirilmiş komut setleridir.
* **`/workflows` (İş Akışları):** Tekrar eden süreçleri otomatize eden; kod inceleme, veritabanı yönetimi, dağıtım (deployment) veya test gibi adımları uçtan uca standartlaştıran iş akışı şablonlarıdır.

## ⚙️ Kurulum ve Kullanım

Buradaki yapılandırmaları kendi Antigravity çalışma alanınıza (workspace) entegre etmek için aşağıdaki adımları izleyebilirsiniz:

1. Bu depoyu yerel ortamınıza klonlayın:
   ```bash
   git clone [https://github.com/baybars03/antigravity-egitimi.git](https://github.com/baybars03/antigravity-egitimi.git)

   
   Antigravity IDE'yi açın.

Klonladığınız dizindeki rules, skills ve workflows dosyalarını, mevcut projenizin yapılandırma klasörüne (genellikle .antigravity veya ilgili konfigürasyon dizini) kopyalayın veya doğrudan içe aktarın.

Değişikliklerin aktif olması için IDE yapılandırmalarını yenileyin (Reload Window).
